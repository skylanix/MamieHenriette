Procédure de configuration de Twitch

Avant toute chose, activez l'authentification à deux facteurs (2FA) : Guide officiel Twitch pour la 2FA

Rendez-vous sur la console d'applications Twitch et ajoutez une application. Renseignez :

  • URL de redirection : https://twitchtokengenerator.com
  • Catégorie : Chat Bot

Créez le bot. Puis, de retour à la liste, éditez-le en cliquant sur Gérer. Puis cliquez sur Nouveau Secret. Vous trouverez ici le Client ID et le Client Secret.

Ensuite, rendez-vous sur twitchtokengenerator.com et sélectionnez Custom Scope Target. Dans la section Use My Client Secret and Client ID, renseignez les deux champs.

Ensuite, dans la section Available Token Scopes, cochez chat:read et chat:edit. Puis, dans la même section, cliquez sur Generate token.

Suivez la procédure et vous avez votre Access Token et Refresh Token
